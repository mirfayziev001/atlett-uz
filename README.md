# ATLETT — сайт магазина

## Что внутри
- `index.html` — главная страница
- `catalog.html` — каталог всех товаров с фильтром по категориям
- `product.html` — карточка товара (открывается как `product.html?id=p001`)
- `admin.html` — **панель управления товарами** (добавление, редактирование, удаление, фото, цены, описания)
- `data/products.json` — файл, где хранятся все товары, цены и тексты. Именно его редактирует `admin.html`
- `assets/` — стили, логотип, картинки-заглушки товаров

Сайт статический — ему не нужен сервер с базой данных. Достаточно обычного хостинга (Hostinger, Beget, GitHub Pages, Netlify, Vercel и т.п.) — просто загрузите туда всю папку `atlett`.

## Как редактировать товары
1. Откройте `admin.html` в браузере (после публикации сайта — по адресу `ваш-сайт.uz/admin.html`).
2. Добавляйте товары, меняйте цены, загружайте фото, пишите описания.
3. Нажмите **«Скачать products.json»** — браузер скачает обновлённый файл.
4. Замените старый файл `data/products.json` на хостинге на скачанный (через файловый менеджер хостинга или FTP-клиент вроде FileZilla).
5. Обновите страницу сайта — изменения появятся.

Важно: `admin.html` ничего не сохраняет автоматически «в облако» — он только готовит файл. Загрузка на хостинг — последний обязательный шаг.

Если не хотите каждый раз перезаливать файл вручную, самый простой вариант — держать `admin.html` открытым только у себя (не давать ссылку клиентам), а `products.json` редактировать хоть вручную в блокноте — это обычный текстовый JSON-файл, там видно `name`, `price`, `description` и так далее.

## Как изменить контакты (Telegram/Instagram) и валюту
Откройте `data/products.json`, в самом начале файла — блок `"shop"`:
```json
"telegram": "atlett.uz",
"telegram_url": "https://t.me/atlett_uz",
"instagram": "atlett_uz",
"instagram_url": "https://instagram.com/atlett_uz",
"currency": "сум"
```
Поменяйте значения и сохраните файл — шапка, футер и кнопки заказа обновятся автоматически на всех страницах.

## Как работает заказ
Кнопка «Заказать в Telegram» на странице товара открывает диалог с вашим Telegram-аккаунтом и **сразу подставляет готовое сообщение** с названием товара, ценой и выбранным размером — покупателю остаётся только нажать «Отправить».

## Публикация сайта
Самый простой бесплатный способ — **GitHub Pages** или **Netlify** (перетащить папку на netlify.com/drop). Для подключения домена `atlett.uz` понадобится настроить DNS-записи у регистратора домена на выбранный хостинг.
